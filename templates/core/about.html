About SkillForge

SkillForge is a Django learning project that feels like a real app: browse curated courses, complete practical challenges, and publish a portfolio you’re proud of.

MVT + SOLID Secure auth & perms CBVs & clean templates

What you’ll learn building SkillForge

MVT done right

Tidy separation across models, views (CBVs), templates, and forms.

Forms & validation

Server-side validation with user-friendly error messages and clean widgets.

Auth & permissions

Public vs private, staff vs superuser, and secure ownership checks.

Query efficiency

Use select_related/annotate to keep pages snappy.

Admin customization

Filters, search, ordering, and safe role management from the admin.

Security basics

CSRF, XSS, parameter tampering prevention, and safe redirects.

How SkillForge is structured

Apps

core (home, about, dashboard), accounts (auth), courses (public CBVs), challenges (private CRUD), portfolio (MAIN feature).

  • At least 10 pages, 5+ CBVs, 5+ models, 5+ forms, responsive templates.
  • Public browse (courses), private CRUD (challenges/submissions, portfolio).
  • Admin split: staff (limited) and superuser (full).
Defense tips
  • Start with a 60-second live demo: courses → challenge → submit → portfolio.
  • Show the code flow: URL → CBV → form → template; highlight validation and messages.
  • Open the admin: show filters/search and role separation.
  • Address security: CSRF in forms, ownership checks, escaping in templates.
